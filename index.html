Projects

RecycleAI

Python

RecycleAI is an AI image classification web application that I built using python. The application featured a user interface to allow people to upload an image or take a picture of an object and then processed that image and classified it as recycle, trash, or compost. The image classification AI was built using Python's TensorFlow library. Using a dataset of over 20,000 images, I trained a model that could differentiate images into different categories of waste. The idea for this project stemmed from the environmental issues in our current society and creating a user-friendly platform to help people sort their waste in an environmentally-safe manner.

WeatherApp

Python

WeatherApp is a personal project I created to retrieve weather info for my local area. I built this project using Python and the BeautifulSoup and Selenium libraries. The program incorporated a web scraper that scanned for the weakly forecast data on Weather.com and was then able to display the data in a user-friendly way using Pandas. Given the desired zip code, the program first used Selenium to automate the process of navigating to the 10-day forecast section of Weather.com. After the URL for the location was received, I used the web scraper to obtain the specific weather info.
